# DigiTools Platform

A modern React + Vite landing page for digital productivity tools. This project showcases premium digital products, a responsive shopping cart experience, and toast notifications for user actions.

## Project Overview

DigiTools Platform is a clean UI product showcase built with React and Tailwind-style utility classes. It includes:

- Product browsing with a responsive grid layout
- Add-to-cart interactions with a live cart count in the navbar
- Cart removal and checkout flows with toast notifications

## Technologies Used

- React 19
- Vite
- React Toastify
- JavaScript
- CSS / Tailwind-style utility classes

## Key Features

1. Add to Cart with toast notifications
2. Remove individual cart items from the cart section
3. Proceed to Checkout clears all cart items and resets the cart count

## Behavior

- Clicking a product's `Buy Now` button adds it to the cart and temporarily changes the button label to `Added to cart`
- The navbar cart icon updates with the current cart count
- The cart page shows selected items, total price, remove buttons, and a checkout button
- Toast alerts appear for add, remove, and checkout actions
